# planet-gravity
Basic 3D planetary gravity simulator, made as a final project for spring 2022 CPSC-4050

## Installing

1. Create the virutal environment with `python3 -m venv .venv`
2. Activate the virtual environment
    - Mac: `source .venv/bin/activate`
    - Windows: `.\venv\Scripts\activate`
3. Install all dependancies with `pip install -r requirements.txt`
    - If a dependancy is added or removed run `pip freeze > requirements.txt` within the virtual environment

## Running

1. Activate the virtual environment
    - Mac: `source .venv/bin/activate`
    - Windows: `.\venv\Scripts\activate`
2. Run the program with `python src/main.py`
